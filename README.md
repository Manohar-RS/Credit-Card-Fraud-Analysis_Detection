# Credit Card Fraud Detection

## Overview

This project detects fraudulent credit card transactions using Machine Learning.

## Dataset

* Total Records: 284,807
* Total Features: 31
* Fraud Transactions: 492
* Normal Transactions: 284,315

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
* SMOTE
* Logistic Regression
* Jupyter Notebook

## Project Workflow

1. Data Loading
2. Data Exploration
3. Data Preprocessing
4. Feature Scaling
5. Handling Class Imbalance using SMOTE
6. Model Training using Logistic Regression
7. Model Evaluation
8. Model Saving using Pickle

## Model Performance

* Accuracy: 97.45%
* Precision: 5.93%
* Recall: 92.86%
* F1 Score: 11.15%
